# NFL-Sentiment-Analysis
Our project is split up into 3 separate sections: Webscraping, Sentiment Analysis, and Regression.

To run our project yourself, please note the following:

1. Each section contains a .py functions file and .ipynb where you can run that section. Howevever, due to processing issues, both the functions and code to run are contained only in a .ipynb folder for the webscraping section. 

Additionally, because the webscraping section can take a very long time to run, its output is also included as CSV files in the folder, so you can also proceed by running the Sentiment folder without the data processing part to save time!
