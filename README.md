# NFL-Sentiment-Analysis
Our project is split up into 3 separate sections: Webscraping, Sentiment Analysis, and Regression.

Group Contributions:
Nicole - Webscraping
Anjali - Sentiment Analysis
Oliver - Regression

Note: We divided up the article collection itself evenly between ourselves. In addition, Anjali and Oliver helped with a portion of the webscraping functions as that was by far the most time intensive task we had to undertake as a group. 


To run our project yourself, please note the following:

1. Each section contains a .py functions file and .ipynb where you can run that section. Howevever, due to processing issues, both the functions and code to run are contained only in a .ipynb folder for the webscraping section. 

2. Because the webscraping section can take 10+ minutes to run, its output is also included as CSV files, so you can run the Sentiment folder notebook independently (without the data processing part) to save time.
